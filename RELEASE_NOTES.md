## 🎉 이번 릴리즈 요약
- 아이콘이 추가되었습니다
- 운동 기록 모달에 세트 간 휴식 타이머 기능이 추가되었습니다

## ✨ 추가된 기능
- **휴식 타이머** — 메인 운동 섹션에서 짧게 탭하면 시작/정지, 길게 누르면(500ms) 시간 설정 시트 오픈
- **프리셋 시간** — 30초 / 1분 / 1분 30초 / 2분 / 3분 (기본값 1분)
- **타이머 컨트롤** — 실행 중 +15초, 일시정지/재개, 정지 버튼 제공
- **종료 알림** — 마지막 3초 카운트다운 소리 + 완료 시 알림음 + 진동
- **알림 설정 토글** — 소리/진동 각각 on/off 가능 (설정 유지)

## 🔧 변경/개선
- 아이콘 Flow Series 형태로 개선
- **Part Distribution 위치 이동** — 대시보드 하단 → 달력 탭 하단으로 이동 (기록 분석은 달력과 함께 보는 게 자연스러움)
- **Part Distribution 집계 범위 확장** — 기존 메인 운동만 집계 → 폼롤러/스트레칭/코어/복근/PT(상체)/PT(하체)/유산소까지 전체 운동 항목 표시
