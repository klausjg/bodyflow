## 🎉 이번 릴리즈 요약
세트 편집을 한 화면에서. 5번 클릭할 걸 1번으로 끝내세요.

## ✨ 추가된 기능
- **그룹 편집 모달**: 한 운동의 모든 세트를 한 번에 수정
- **세트 추가 / 개별 삭제 / 전체 삭제** 지원
- **메모 전용 팝업**: 📝 버튼으로 깔끔하게 편집

## 🔧 변경/개선
- 기록 화면을 **그룹 카드** 형태로 재디자인
- 용어 영문 통일: **Main / Finisher / Daily**
- 같은 날 같은 운동 두 세션은 자동으로 분리 표시

## 🐛 수정된 문제
- 저장 직후 완료 토스트가 가려지던 문제

## 📝 기타
- 기존 한글 기록도 그대로 호환됩니다
